Fun with Programming

präsentiert von Dr. Dr. Sheldon Cooper

Lektionen

Steigen Sie ein in die abenteuerliche Welt von 'Hello World' oder ist der Nervenkitzel der aufregenden Entdeckungsreise einer Auszeichnungssprache, begleitet von Tags, Standalones, Attributen und Elementen, etwas für Sie? Denn denken Sie daran, was ein großer Mann einst sagte:
"Everbody in this country should learn how to program a computer...it teaches you how to think."
-Steve Jobs Pilot

In einer kalt-feuchten Winternacht im Jahre 1943 wachte Augusta Ada Lovelace, geborene Bryon mit einer Idee auf, die das folgende Zeitalter nachhaltig prägen würde. Die britische Mathematikerin beschrieb in ihren mit einem Füllhalter geschriebenen “Notes” eine Möglichkeit zur Berechnung von Bernoulli-Zahlen, eine Fortführung von Babbages und Menabrea's Arbeiten – für eine Maschine, die nie gebaut wurde.

Sie träumte von einer Maschine, die nicht nur einfache Rechenaufgabe erfüllen könnte, sondern Buchstaben verarbeiten oder sogar Meisterwerke komponieren würde.

Sie alle folgten einer Bestimmung, die späteren Generationen den Weg für eine neue Form von Gesellschaft und Kommunikation ebnen würden.

Folge 1 - Lovelace, ein neues Gedankengut oder kindische Tagträumerei?
